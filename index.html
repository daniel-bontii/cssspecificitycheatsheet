CSS Specificity Cheatsheat

Selector Declaration Application
Type type {
declaration....
}		 Applies style to the selected type. For instance p, h1, table. Remember not to add the angle brackets to the type
Class .class {
declaration....
}		 Applies style to the all types with a class name. For instance <p class="class"> This paragraph will get the styles in .class ruleset </p>
ID #Idname {
declaration....
}		 Applies style to the all types with an id name. For instance <p id="Idname"> This paragraph will get the styles in #id ruleset </p>
Attribute type[attribute] {
declaration....
}		 Applies to elements based on the presence or value of a given attribute. a[title] { color: purple; }, an <a> element with a title attribute gets the color purple
Things to note
  • Order of increasing specificity goes from Type to Class and finally to ID
  • Classes override type and IDs override classes!
